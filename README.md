# Snoopy’s Chat

A lightweight, embeddable web chat built with plain HTML, CSS, and JavaScript.  
Designed to work smoothly on **GitHub Pages**, embeds, and hub-style tool sites like **Snoopy’s Hub**.

---

## Features

- Full-screen responsive layout
- No frameworks — pure HTML/CSS/JS
- Easy to embed with an `<iframe>`
- Works on GitHub Pages
- CDN-friendly
- Custom glass/blur style support
